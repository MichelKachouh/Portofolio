# Mini Portfolio Homepage

A responsive portfolio homepage built with HTML, CSS, and JavaScript as part of my web development journey

## Overview

This project was created to practice fundamental front-end development skills before learning CSS Grid. The focus was on building a clean, responsive layout using semantic HTML, Flexbox, and basic JavaScript.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Hero section with introduction
* About section
* Projects showcase section
* Skills section
* Contact form layout
* Dark/Light mode toggle
* Clean and modern user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox

## Learning Objectives

Through this project, I practiced:

* Writing semantic HTML
* Creating responsive layouts
* Using Flexbox for positioning and alignment
* Structuring multi-section webpages
* Working with CSS styling and spacing
* Manipulating the DOM with JavaScript
* Implementing a theme toggle feature

## Challenges

Some of the challenges faced during development included:

* Maintaining consistent spacing throughout the page
* Creating a responsive layout for different screen sizes
* Organizing CSS efficiently
* Implementing JavaScript interactions without overcomplicating the project

## Future Improvements

* Rebuild the layout using CSS Grid
* Add animations and transitions
* Improve accessibility
* Add project filtering functionality

## Lessons Learned

This project strengthened my understanding of HTML structure, CSS layout techniques, responsive design principles, and JavaScript DOM manipulation. It also provided valuable experience planning and building a complete webpage from scratch.
